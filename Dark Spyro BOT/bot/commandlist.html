Welcome to DS003_Bot, a bot designed by Dark_Spyro_003 alias Sparx & Marforius. Here is the commands list:
Command type    :: Command      arguments     :: Command info
Privmsg Command :: help                       :: shows this help text
Privmsg Command :: identify     (password)    :: identifies as bot operator
_____
Channel Command :: =disconnect                :: Requires bot operator :: Disconnects the bot from IRC
Channel Command :: =nickregist                :: Requires bot operator :: Registers the bot with NickServ
Channel Command :: =nickident                 :: Requires bot operator :: Identifies the bot with NickServ
Channel Command :: =join        (channelnames):: Requires bot operator :: Joins channel name (channelname)
Channel Command :: =leave       (channelnames):: Requires bot operator :: Leaves channel name (channelname)
Channel Command :: =say         (text)        :: Requires bot operator :: Says text (text)
Channel Command :: =action      (action)      :: Requires bot operator :: Performs emote (action)
_____
Channel Command :: ==projectname (subcommand)  :: Shows or links to requested info of project. I.e. ==arc trac.
Channel Command :: =time        (option pro)  :: Displays the current time (optional with date).
Channel Command :: =npc         (id or name)  :: Links to an NPC with entry (id) or name (name) on WoWhead
Channel Command :: =item        (id or name)  :: Links to an item with entry (id) or name (name) on WoWhead
Channel Command :: =spell       (id or name)  :: Links to a spell with entry (id) or name (name) on WoWhead
Channel Command :: =achievement (id or name)  :: Links to an achievement with entry (id) or name (name) on WoWhead
Channel Command :: =faction     (id or name)  :: Links to a faction with entry (id) or name (name) on WoWhead
Channel Command :: =object      (id or name)  :: Links to an object with entry (id) or name (name) on WoWhead
Channel Command :: =skill       (id or name)  :: Links to a skill with entry (id) or name (name) on WoWhead
Channel Command :: =title       (id or name)  :: Links to a title with entry (id) or name (name) on WoWhead
Channel Command :: =zone        (id or name)  :: Links to a zone with entry (id) or name (name) on WoWhead
Channel Command :: =class       (id or name)  :: Links to a class with entry (id) or name (name) on WoWhead
Channel Command :: =race        (id or name)  :: Links to a race  with entry (id) or name (name) on WoWhead
_____
Channel Command :: =mailsend (nick) (message) :: Sends a botmail to nickname (nick) with message (message)
Channel Command :: =mailcheck                 :: Checks for new botmail.
Channel Command :: =maildelete                :: Deletes your botmail.
